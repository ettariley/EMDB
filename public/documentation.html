EMDB (Etta's Movie Database)

This is a REST API for an application called EMDB that stores data about different types of movies.

URL Endpoints

Request URL HTTP Method Request Data Format Response Data Body Format
Return a list of all movies to the user /movies GET None A JSON object holding data about all movies (See row below for movie format example)
Get data about a single movie by title /movies/[TITLE] GET None A JSON object holding data about a single movie, containing a description, genre, director, image URL, and whether it's featured or not. Example:
{
"Title":"JoJo Rabbit",
"Description":"A young German boy in the Hitler Youth whose hero and imaginary friend is the country's dictator is shocked to discover that his mother is hiding a Jewish girl in their home.",
"Genre": {
"Name":"Comedy",
"Description":"Funny, humorous, satirical, or lighthearted fiction or non-fiction movies."
},
"Director": {
"Name":"Taika Watiti",
"Bio":"Taika hails from the Raukokore region of the East Coast of New Zealand. He has been in the film industry for several years as an actor, writer, and director. He is most known for his work directing Thor: Ragnarok (2017) and JoJo Rabbit (2019), for which he won an Oscar for Best Adapted Screenplay.",
"Birth":"1975-01-01"
},
"ReleaseYear": "2019-01-01" "ImageURL":"jojorabbit.png",
"Featured":false
}
Get data about a single genre by name genre/[NAME] GET None A JSON object holding data about a single genre, including its description. Example:
{
"Name":"Comedy", "Description":"Funny, humorous, satirical, or lighthearted fiction or non-fiction movies."
}
Get data about a single director by name /director/[NAME] GET None A JSON object holding data about a single director, containing a name, bio, birth year, and death year if one. Example:
{
"Name":"Taika Watiti",
"Bio":"Taika hails from the Raukokore region of the East Coast of New Zealand. He has been in the film industry for several years as an actor, writer, and director. He is most known for his work directing Thor: Ragnarok (2017) and JoJo Rabbit (2019), for which he won an Oscar for Best Adapted Screenplay.",
"Birth":"1975-01-01"
}
Add a user /users POST A JSON object holding data about the user to add, formatted like so:
{
"Username": "etta" (required)
"Password": "pass1234" (required)
"Email": "example@example.com" (required)
"Birthday": 1993-07-02 (required)
"FavoriteMovies": []
} 		A JSON object holding data about the user that was added, including an ID. Example:
{
_id: ObjectID("1234"), "Username": "Etta"
"Password": "pass1234"
"Email": "example@example.com"
"Birthday": 1993-07-02
"FavoriteMovies": []
}
Update user's info /users/[Username] PUT A JSON object holding the data to be updated. Example:
{
"Username": "Etta2"
} 		A JSON object of the user information (See above for example format)
Add movie to user's list of favorites /users/[Username]/movies/[MovieID] POST None A JSON object with the user information.
Remove movie from user's list of favorites /users/[Username]/movies/[MovieID] DELETE None A JSON object with the user information.
Remove user /users/[Username] DELETE None A text message indicating whether the user was successfully deregistered.
